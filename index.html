Bond's stress in a shockless high-pulsed power spalling simulation

Codes
GranOO / Python / Paraview
Project
PhD
Details
An AL23 ceramic target is modeled with the Discrete Element Method (DEM). An experimental pressure pulse is applied to its front face. The reflection of the first incident compressive stress wave on the rear face interacts with the second and generates tensile in the target. The tensile stress state leads to fragmentation - spalling - by bond removal.

Bond's stress in a Edge-On-Impact (EOI) simulation

Codes
GranOO / Python / Paraview
Project
PhD
Details
An AL23 ceramic target is modeled with the Discrete Element Method (DEM). A steel projectile is impacted on the edge of the target. Orthoradial and spalling cracks appear due to tensile stress state. The impact zone is completely fragmented.
